STEPS TO RUN THE ML-SERVER

1) Create the .venv by running this below cmd,
        python -m venv .venv

2) Switch to ml-server by using below cmd
        cd ml-server

3) Active the .venv by using the below cmd,
        .venv/scripts/activate

4) To run 
        python app.py

5) Open http://127.0.0.1:5000/
